# Node.js

## 安装

本项目最低支持 22 版本的 Node.js

下载地址：https://nodejs.org/zh-cn/download

## Node.js 版本管理工具

可能你需要同时使用多个版本的 Node.js；这里推荐你使用 node 版本管理工具 N。

> https://www.npmjs.com/package/n

### 安装 N

```shell
sudo npm install -g n
```

### 安装指定版本的 Node.js

```shell
sudo n v22.21.1
```

### 切换 Node.js 版本

```shell
sudo n
```

### 卸载指定版本的 Node.js

```shell
sudo n rm v14.18.3
```
